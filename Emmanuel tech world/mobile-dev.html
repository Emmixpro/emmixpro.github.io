Frequently Asked Questions

Which stack will we focus on - React Native or Flutter? +

We primarily focus on React Native as it's JavaScript-based and has a gentler learning curve. However, we include Flutter overview sessions and compare both ecosystems so you can make informed decisions for future projects.

Do I need a Mac for iOS development? +

For full iOS publishing, yes - you'll need a Mac. However, you can develop and test iOS apps using Expo and simulators during the course. We provide alternatives for Windows/Linux users and focus on techniques that work across platforms.

What about prior programming experience? +

Basic programming knowledge is helpful but we start from fundamentals. If you're completely new to programming, we recommend going through our JavaScript basics resources first (provided to all students).

Will I be able to get a job after this course? +

Yes! We focus on practical, employable skills. You'll build a portfolio of 3 real apps, learn industry best practices, and receive career guidance. Many of our students secure junior positions or freelance work upon completion.

What support is available during the course? +

You get full support: weekly live Q&A sessions, dedicated Telegram group for daily questions, code reviews, and 1:1 mentorship sessions. Our instructors are actively involved throughout your learning journey.

Are the classes recorded? +

Yes! All live sessions are recorded and available within hours. You get lifetime access to all course materials, updates, and recordings, so you can learn at your own pace.
